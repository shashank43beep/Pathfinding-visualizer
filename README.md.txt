# Pathfinding Visualizer

A Python + Pygame project that visualizes pathfinding algorithms:

- BFS
- DFS
- A*

## Features
- Interactive grid
- Random maze generation
- Visual shortest path

## Controls
- Left Click: Place start/end/walls
- Right Click: Remove
- SPACE: Run algorithm
- B/D/A: BFS/DFS/A*
- C: Clear
- R: Random maze

## Run
```bash
python main.py
